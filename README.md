# Common development processes:
---

- Install dev and test requirements
- Install pre-commit hooks: `pre-commit install`
- Write source code and test code at `src` and `tests` respectively
- Run pre-commit hooks: `pre-commit run --all-files`
- For testing, simply run `pytest` or run from vscode
- Use vscode interactive debugging
- Install this project in "edit" mode: `pip install -e .`
- Run type checker: `mypy .`
